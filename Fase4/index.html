Propósito del sitio

El propósito de este sitio web es de utilizar técnicas de codificación HTML y la aplicación de los estilos CSS.

Tambien se implementara el uso del Felx Box o Modulos de Caja Flexible, las Listas y los Enlaces.

Introducción

La utilización de los layout es muy vital para la construcción de las paginas web creadas por medio de HTML y CSS, un metodo más común es la utilización de los Flex Box , Listas y Enlaces. los Flex Box es el que nos permite maquinar nuestras paginas web de una manera mucho más fácil como se hacia anteriormente, se le llama Flex porque se tiene un contenedor, en la que se contiene la propiedad display: flex.

Los Enlaces, son como atajos a archivos dentro del mismo documento, equipo o en otro servidor. Por ultimo las Listas, estas nos permiten crear conjuntos de elemtos en forma de lita dentro de nuestra pagina que generalmente se encuentra procedido por un número o guíon.
