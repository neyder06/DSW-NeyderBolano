Flex Box

El Flex Box es una propiedad de CSS display, el cual nos permite maquinar nuestras paginas web en una forma más fácil y sencilla distintas como sr hacian en tiempos anteriores. Se le llama Flex porque se tiene un contenedor que contiene la propiedad display: flex.



File:Exemplos de Flexbox.png. (s/f). Wikimedia.org. Recuperado el 19 de septiembre de 2021, de https://commons.wikimedia.org/wiki/File:Exemplos_de_Flexbox.png

En este contenedor vamos a poder especificar, una alineación de los elementos que se encuentra dentro de este contenedor un tamaño que contiene y que pueda distribuir un espacion restante que hay entre los elementos de nuestro contenedor.

  • Se puede hacer distintos diseños flexibles que utilicen menos código fuente, estos nos permiten optimizar nuestro código y nuestro sitio web.
  • Se puede alinear los elementos en una sola dirección, ya sea horizontal o vertical, con esta alineación nos ayudara a distribuir nuestros elementos sin que coloquemos nada dentro de ellos.
  • Se puede reordenar los contenidos sin tocar nuestro código fuente (HTML), el cual nos permitirá optimizar nuestra página web.
